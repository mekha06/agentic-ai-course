# Module 2 - Tools and Function Calling

## Goal

In this module, I learned how AI agents use external tools to perform real actions.

An LLM alone can only generate text.  
An AI Agent becomes powerful when it can use tools such as calculators, APIs, databases, search engines, or custom Python functions.

---

## Topics Covered

- Why agents need tools
- What is a tool in Agentic AI
- Difference between a function and a tool
- Tool selection
- Function calling
- Structured outputs
- Building a mini tool-using agent
